RECOLECTOR DE DATOS DE PLC'S Y SU VISUALIZACIÓN EN INTERFAZ GRÁFICA  

Este proyecto fue desarrollado como parte de mis estadías profesionales, utilizando Angular y MongoDB. Todo el código, la base de datos y el diseño fueron creados íntegramente por mí desde cero.  

DESCRIPCIÓN DEL PROYECTO  
El objetivo de este proyecto fue disminuir la necesidad de técnicos especializados en PLCs mediante la creación de un software capaz de extraer información de los PLCs a través de MODBUS y TCP/IP. Los datos recopilados se almacenan en una base de datos MongoDB y se pueden visualizar a través de una interfaz gráfica desarrollada en Angular.  

TECNOLOGÍAS UTILIZADAS  

- Angular: Para el desarrollo del frontend.  
- MongoDB: Para la base de datos (creada en mi cuenta personal).  
- MODBUS: Para la comunicación con los PLCs.  
- TCP/IP: Para la comunicación con los PLCs.  

TAREAS REALIZADAS  

- Manejo de base de datos NoSQL (MongoDB).  
- Uso de PLC's.  
- Creación de software con interfaz gráfica.  

NOTAS IMPORTANTES  

- Este proyecto no contiene información confidencial ni datos sensibles de la empresa.  
- La base de datos usada para pruebas fue creada en mi cuenta personal de MongoDB y ya no está vinculada a ningún entorno empresarial.  
- Todos los datos usados durante el desarrollo fueron únicamente de prueba.  
- El programa contiene archivos de pruebas usados durante el proceso de desarrollo.  

CÓMO EJECUTAR EL PROYECTO  

Clona el repositorio:  

git clone ""  

Instala las dependencias:  

npm install  

Levanta el servidor de desarrollo:  

ng serve  

Abre el navegador en http://localhost:4200/

PLC DATA COLLECTOR AND GRAPHICAL INTERFACE VISUALIZATION

This project was developed as part of my professional internship, using Angular and MongoDB. All the code, database, and design were entirely created by me from scratch.

PROJECT DESCRIPTION  
The goal of this project was to reduce the need for specialized PLC technicians by developing software capable of extracting information from PLCs using MODBUS and TCP/IP. The collected data is stored in a MongoDB database and can be visualized through a graphical interface developed in Angular.

TECHNOLOGIES USED  

- Angular: For frontend development.  
- MongoDB: For the database (created in my personal account).  
- MODBUS: For communication with PLCs.  
- TCP/IP: For communication with PLCs.  

TASKS PERFORMED  

- Management of NoSQL databases (MongoDB).  
- Working with PLCs.  
- Development of software with a graphical interface.  

IMPORTANT NOTES  

- This project does not contain confidential or sensitive company data.  
- The database used for testing was created in my personal MongoDB account and is no longer linked to any corporate environment.  
- All data used during development was for testing purposes only.  
- The program includes test files used during the development process.  

HOW TO RUN THE PROJECT  

Clone the repository:  

git clone ""  

Install dependencies:  

npm install  

Start the development server:  

ng serve  

Open your browser at http://localhost:4200/
